# CodeAlpha-_-Secure-coding-review
Secure Coding Review - Python Login System

Project Overview

This project is a Secure Coding Review of a Python Login System using Tkinter and MySQL.


Objectives


Review source code

Identify vulnerabilities

Suggest secure coding practices

Recommend fixes


Vulnerabilities Found


Plain text password storage

Hardcoded credentials

Weak input validation

Missing brute-force protection

Poor exception handling


Tools Used


Manual Code Review

Bandit

Pylint

SonarQube


Recommendations


Use password hashing

Add input validation

Store secrets securely

Add login restrictions


Conclusion

The application requires several security improvements before production use.
